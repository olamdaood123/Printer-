### 📝 عن المشروع (About)
تطبيق مكتبي متطور تم بناؤه باستخدام لغة #C وبيئة .NET Framework. يهدف المشروع لتسهيل عمليات طباعة الوثائق والهويات مع ميزات معالجة صور ذكية تمنع الأخطاء الشائعة في القص.

---

### 🚀 المميزات الاحترافية (Professional Features)

* 🎨 خوارزمية فحص الألوان: 
    * البرنامج مزود بذكاء يمنع قص الهوية تلقائياً إذا كانت الخلفية تحتوي على تدرجات لونية متعددة، لضمان دقة الصورة الأصلية.

* 📲 طرق الإرسال والتحميل: 
    * تحميل مباشر: إمكانية رفع الصور والوثائق من ملفات الجهاز.
    * WhatsApp: أزرار مخصصة للارتباط السريع مع تطبيق واتساب لمشاركة المستندات.
    * Bluetooth: دعم إرسال أوامر الطباعة عبر البلوتوث.

* ⚙️ إدارة أدوات الطباعة: 
    * واجهة سهلة للتحكم في عدد النسخ وإعدادات الورق والمعاينة الدقيقة قبل الطباعة.
